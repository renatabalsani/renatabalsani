Sobre mim

O meu nome é Renata Balsani

Graduada em Tecnologia da Informação, Engenharia da Computação e Recursos Humanos, sou Analista de Dados Sênior com mais de 20 anos de experiência nas áreas de Análise de Dados, Financeira, Contas a Pagar e Recebimentos.

Tenho habilidades consolidadas em controle, gestão financeira e criação de visualizações analíticas, além de capacidade para resolver problemas complexos com uma abordagem analítica, focada em soluções criativas e de alto impacto, utilizando ferramentas como Python, Power BI, Excel, SQL Server e BigQuery.

Minha experiência inclui a criação de relatórios focados na jornada do usuário e o uso de storytelling, juntamente com o conhecimento de linguagens de análise de dados capazes de gerar insights estratégicos para o processo de tomada de decisão.

Projetos de análise de dados aplicados

Projeto de Análise de Dados da Fórmula 1: Desempenho de Equipes, Pilotos e Corridas.

Desenvolvi um projeto de análise de dados para a Fórmula 1, com o objetivo de examinar o desempenho das equipes, dos pilotos e das corridas individuais. O projeto inclui a análise das estatísticas das corridas, a comparação entre o desempenho dos pilotos e a eficácia das equipes em diferentes temporadas. Uma novidade interessante é a inclusão de um vídeo de cobertura no dashboard, que inseri utilizando HTML e GitHub.

Ferramentas Utilizadas:

  • Power BI: para a criação de dashboards interativos;
  • Figma: para o design e a concepção da interface do usuário;
  • HTML: para a integração de um vídeo de cobertura no dashboard;
  • GitHub: para o gerenciamento do projeto e compartilhamento do código.
